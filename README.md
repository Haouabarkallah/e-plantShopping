
#  e-plantShopping — Paradise Nursery



##  Project Overview

**e-plantShopping** is a front-end shopping application for **Paradise Nursery** — a houseplant e-commerce platform. Built with React, this app allows users to browse a curated collection of indoor plants, add them to a shopping cart, and manage quantities before checkout. The application features a clean, intuitive interface with three main pages: a welcoming landing page, a categorized product listing, and a fully interactive shopping cart.

### Key Features

####  Landing Page
- Visually appealing **background image** that reflects the botanical theme
- A brief **company description** about Paradise Nursery's mission to bring nature indoors
- **Company name** prominently displayed
- A **Get Started** button that navigates to the product listing page

####  Product Listing Page
- Displays **6+ houseplants** organized into **3+ categories** (e.g., Indoor Favorites, Air Purifying Plants, Low Maintenance)
- Each plant card includes:
  - Thumbnail image
  - Plant name
  - Price
  - **Add to Cart** button
- **Persistent header** featuring:
  - Shopping cart icon with a **dynamic badge count** showing total items in cart
  - Navigation links to switch between pages

#### Shopping Cart Page
- **Summary section** displaying:
  - Total number of plants in the cart
  - Total cost of all items
- **Per-item view** showing:
  - Thumbnail and plant name
  - Unit price and line total
  - **Increase/Decrease** buttons to adjust quantity (updates badge and totals in real-time)
  - **Delete** button to remove the item entirely
- **Continue Shopping** button to return to the product listing
- **Checkout** button
